# Supply Chain Operations Analysis Dashboard

## Project Overview

This project analyzes a supply chain dataset to understand product performance, supplier efficiency, logistics costs, and quality issues.
The goal is to extract meaningful insights from operational data and visualize them using an interactive dashboard.

The analysis helps answer key business questions such as:

* Which product categories generate the most revenue?
* Which suppliers contribute the most to sales?
* Which transportation methods are the most expensive?
* Which suppliers have higher defect rates?

---

## Tools Used

* Excel – Data cleaning and preparation
* MySQL – Data storage and SQL analysis
* Power BI – Dashboard visualization and reporting

---

## Dataset

The dataset contains supply chain information including:

* Product type
* Supplier name
* Number of products sold
* Revenue generated
* Shipping costs
* Transportation modes
* Routes
* Manufacturing costs
* Defect rates
* Lead times and stock levels

---

## Analysis Performed

Using SQL and Power BI, the following analyses were conducted:

* Revenue analysis by product category
* Supplier revenue performance analysis
* Product demand analysis based on units sold
* Transportation cost comparison across shipping modes
* Route cost comparison for logistics optimization
* Supplier quality analysis based on defect rates

---

## Dashboard Insights

Key insights from the dashboard include:

* Certain product categories contribute significantly more to overall revenue.
* A small number of suppliers account for a large portion of total revenue.
* Transportation modes vary significantly in shipping cost efficiency.
* Some suppliers show higher defect rates, indicating potential quality risks.
* Logistics routes differ in cost, highlighting potential optimization opportunities.

---

## Dashboard Features

The Power BI dashboard includes:

* Key performance indicators (KPIs) such as Total Revenue, Units Sold, Average Shipping Cost, and Average Defect Rate
* Interactive filters for product type, supplier name, and transportation mode
* Visual analysis of supplier performance, logistics cost, and product demand
